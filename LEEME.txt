# PRUEBA DE DIAGNOSTICO

Se deberá desarrollar un Sistema de Votación. La interfaz deberá validar y
guardar los datos del formulario en una base de datos

## Requisitos 
1. PHP Version 8
2. Base de datos MySql

## Instalación

Para instalar el proyecto, sigue los siguientes pasos:

1. Clona el repositorio "git clone https://github.com/JJackDiaz/pruebadesis.git".
2. Crea la base de datos.
3. Ejecuta el scripts ubicado en SQL.
4. Modifica la conexion a la base de datos "config/conexion.php".

## Uso

Para utilizar el proyecto, sigue los siguientes pasos:

1. Abre un navegador web y navega hacia `http://localhost:3000`.
2. Utiliza la aplicación según sea necesario.
